# Watson 日程日历 Web 应用设计文档

## 1. 项目概述

**项目名称**：Watson\
**类型**：纯前端 Web 应用（无后端依赖）\
**运行方式**：浏览器本地运行（支持离线使用）

### 1.1 核心目标

实现一个本地运行的日程管理日历，支持：

-   日历视图（按月展示）
-   日程的 添加 / 修改 / 删除
-   数据本地持久化（刷新或关闭浏览器后仍存在）
-   简洁直观的交互（点击日期、浮动按钮操作）

------------------------------------------------------------------------

## 2. 技术选型建议

## 4. 功能模块

### 4.1 日历模块

-   显示当前月份
-   支持切换年月

``` ts
type Schedule = {
```

## 6. 数据存储

使用 localStorage：

``` ts
const STORAGE_KEY = "watson_schedules";

export function save(data) {
  localStorage.setItem(STORAGE_KEY, JSON.stringify(data));
}

export function load() {
  return JSON.parse(localStorage.getItem(STORAGE_KEY) || "[]");
}
```
